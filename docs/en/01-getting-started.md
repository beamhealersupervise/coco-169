# Quick Start Guide

This guide will help you get started with Coco in 5 minutes.

## Prerequisites

- Go 1.21 or higher
- An OpenAPI/Swagger specification file (JSON format, can be generated by tools like Swag and Huma)

## Step 1: Installation

Install Coco using `go get`:

```bash
go get github.com/leehainuo/coco
```

## Step 2: Prepare OpenAPI Specification

You need an OpenAPI specification file. If you don't have one, create a simple example:

**openapi.json**
```json
{
  "openapi": "3.0.0",
  "info": {
    "title": "Hello API",
    "version": "1.0.0",
    "description": "A simple example API"
  },
  "paths": {
    "/hello": {
      "get": {
        "summary": "Say hello",
        "responses": {
          "200": {
            "description": "Success",
            "content": {
              "application/json": {
                "schema": {
                  "type": "object",
                  "properties": {
                    "message": {
                      "type": "string",
                      "example": "Hello, World!"
                    }
                  }
                }
              }
            }
          }
        }
      }
    }
  }
}
```

## Step 3: Write Code

Create a `main.go` file:

```go
package main

import (
    "net/http"
    "github.com/leehainuo/coco"
)

func main() {
    // Create documentation handler
    handler := coco.New("./openapi.json",
        coco.Title("Hello API Documentation"),
        coco.Lang("en"),
    )
    
    // Mount to /docs/ path
    http.Handle("/docs/", handler)
    
    // Start server
    println("Documentation server started at http://localhost:8080/docs/")
    http.ListenAndServe(":8080", nil)
}
```

## Step 4: Run

```bash
go run main.go
```

Open your browser and visit http://localhost:8080/docs/ to see the beautiful API documentation!

## FAQ

### Documentation page shows blank?

Check the following:
1. Is the OpenAPI file path correct?
2. Is the OpenAPI file format valid? (Use an online validator)
3. Are there any errors in the browser console?

### How to customize the documentation path?

```go
// Path is determined when you mount the handler, no default value
http.Handle("/docs/", handler)      // Use /docs/
http.Handle("/api-docs/", handler)  // Use /api-docs/
http.Handle("/swagger/", handler)   // Use /swagger/
```

### Which OpenAPI versions are supported?

Coco supports:
- OpenAPI 3.0.x
- OpenAPI 3.1.x
- Swagger 2.0

### How to use in production?

It's recommended to disable debug features:

```go
handler := coco.New("./openapi.json",
    coco.EnableDebug(false),
)
```

## 💡 Tips

- Use `coco.Theme("dark")` to set default dark theme
- Use `coco.Lang("en")` for English interface
- Theme and language can be switched in the top-right corner
- Debug panel allows direct API testing
